# 操作系统课程设计

基于 C++ 实现的操作系统核心算法集合，涵盖地址转换、进程调度、内存管理等多个经典操作系统问题。

## 📚 功能模块

本项目包含以下五大核心算法模块：

### 1. 地址转换（分页/分段）
- 支持分页存储管理方式的地址转换
- 支持分段存储管理方式的地址转换
- 可视化展示页表和段表

### 2. 银行家算法（死锁避免）
- 实现银行家算法模拟
- 安全状态检测
- 资源请求与分配
- 动态展示资源分配过程

### 3. 磁盘调度算法
- FCFS（先来先服务）
- SSTF（最短寻道时间优先）
- SCAN（电梯算法）
- CSCAN（循环扫描）

### 4. 页面置换算法
- OPT（最佳置换算法）
- FIFO（先进先出）
- LRU（最近最少使用）
- CLOCK（时钟置换）

### 5. 进程调度算法
- FCFS（先来先服务）
- SJF（短作业优先）
- Priority（优先级调度）
- RR（时间片轮转）

## 🛠️ 技术栈

- **编程语言**: C++
- **编译器**: g++ (支持 C++11 标准)
- **开发环境**: Windows, Visual Studio Code
